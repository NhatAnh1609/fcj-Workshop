### Kết quả đạt được tuần 3:

* Hiểu được IAM Role cho EC2:
  * Tạo EC2 Instance, S3 bucket
  * Tạo và sử dụng IAM user, access key, IAM role
* Hiểu được AWS Cloud9:
  * Tạo Cloud9 instance
  * Sử dụng command line và AWS CLI
* Hiểu được hosting Website tĩnh với Amazon S3:
  * Bật static website hosting
  * Tăng tốc với CloudFront
  * Bật Bucket Versioning và sao chép Object sang region khác
* Hiểu được Amazon RDS:
  * Tạo RDS database instance
  * Backup và restore

### Week 3 Objectives:

* Learn about granting permissions to applications with IAM Roles.
* Learn about AWS Cloud9.

### Week 3 Outcomes:

* Understand IAM Roles for EC2:
  * Launch EC2 instances and create S3 buckets
  * Create and use IAM users, access keys, and IAM roles
* Understand AWS Cloud9:
  * Create Cloud9 environments
  * Use the command line and AWS CLI
* Understand hosting a static website with Amazon S3:
  * Enable static website hosting
  * Speed up content with CloudFront
  * Enable Bucket Versioning and copy objects across regions
* Understand Amazon RDS:
  * Create RDS database instances
  * Perform backup and restore

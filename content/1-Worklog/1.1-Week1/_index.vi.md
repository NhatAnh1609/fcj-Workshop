### Kết quả đạt được tuần 1:

* Hiểu được AWS là gì và nắm được các dịch vụ cơ bản.
* Tạo và cấu hình AWS Free Tier account thành công.
* Hoàn thành 5 nhiệm vụ lấy $200 credit:
  * Task Launch an instance using EC2
  * Task Use a foundation model in the Amazon Bedrock
  * Task Set up a cost budget using AWS Budgets
  * Task Create a web app using AWS Lambda
  * Task Create an Amazon RDS Database
* Tìm hiểu về quản lý chi phí với AWS Budgets:
  * Tạo Cost Budget
  * Tạo Usage Budget
  * Tạo RI Budget
  * Tạo Saving Plans Budget
  * Dọn dẹp tài nguyên

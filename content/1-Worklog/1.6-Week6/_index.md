### Week 6 Objectives:

* Learn how to build highly available web applications.
* Learn AWS Lambda serverless patterns.
* Learn VM migration with AWS VM Import/Export.
* Learn database migration with AWS DMS and SCT.
* Learn disaster recovery with AWS Elastic Disaster Recovery.
